Introduction to Programming
---------------------------
Pemrograman merupakan proses membuat serangkaian instruksi kepada mesin atau komputer untuk melakukan suatu tugas tertentu.

Pertama seseorang akan menulis kode (serangkaian huruf, angka, dan karakter lainnya), lalu compiler akan mengkonversi setiap baris kode ke dalam bahasa yang dapat di mengerti oleh komputer. 
Komputer akan memindai kode lalu menjalankannya sehingga dapat melakukan serangkaian tugas. 

Introduction to Java 
---------------------
Definition
~~~~~~~~~~
Java merupakan salah satu bahasa Pemrograman yang populer untuk membuat komponen back-end dari program komputer, aplikasi Android, dan website.

Dikenal dengan : **Write once run every where**

dijalankan di berbagai platform tanpa perlu mengubah platformnya. Misalnya, beroperasi pada sistem operasi Android, Linux, Windows, dan lainnya.
hal ini dikarenakan Java merupakan *High-Level Programming Languange*

Java History
~~~~~~~~~~~~
Bahasa pemrograman Java pertama kali digunakan pada **"The Green Project"** di Sun Microsystem, sebuah perusahaan perangkat lunak di Amerika Serikat. 

.. image:: /images/01/java.png
   :width: 200

.. note:: 
    Ringkasan sejarah:

    - **Pembuat Proyek**: James Gosling, Patrick Naughton, Mike Sheridan, dan Bill Joy
    - **Tujuan Awal** : membuat perangkat pintar
  

Tujuan awal proyek untuk membuat perangkat pintar. Tapi mereka tidak puas dengan hasil C++ dan C, jadi mereka membuat bahasa pemrograman sendiri yang lebih canggih. 
James Gosling akhirnya membuat bahasa pemrograman baru yang disebut Oak setelah 18 bulan proyek. Sayangnya, perusahaan Oak Technology telah menggunakan nama itu. 
Oleh karena itu, pada tahun 1995, Oak digantikan oleh Java, yang dikatakan terinspirasi oleh kopi Jawa. Akibatnya, logo bahasa pemrograman Java adalah secangkir kopi.

Introduction to IDE
-------------------
**IDE (Integrated Development Environment)**, merupakan sebuah workspace untuk programmer untuk mengembangkan, mengedit, dan menjalankan code nya.

.. image:: /images/01/eclipse.png
   :width: 300

Eclipse merupakan salah satu IDE yang dibuat dari bahasa pemrograman Java,akan tetapi akan tetapi Eclipse mendukung pengembangan aplikasi berbasis bahasa pemrograman lainnya, seperti C/C++, Cobol, Python, Perl, PHP, dan lain sebagainya

Contoh IDE lain selain eclipse yaitu Netbeans, Visual Studio Code, IntelliJ IDEA, BlueJ, JCreator, JDeveloper
